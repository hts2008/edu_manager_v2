# SEO Fundamentals

## On-Page SEO
- Title tag: 50-60 chars, keyword near start
- Meta description: 150-160 chars, include CTA
- H1: one per page, contains primary keyword
- URL structure: short, readable, keyword-included

## Technical SEO
- sitemap.xml: auto-generated, submitted to Search Console
- robots.txt: allow crawling of important pages
- Canonical URLs: prevent duplicate content
- HTTPS: required for ranking
- Mobile-friendly: responsive design required

## Performance SEO
- Page speed: Core Web Vitals as ranking signal
- Server response time under 200ms
- Lazy loading images below fold

## Structured Data
- JSON-LD format preferred
- Schema.org vocabulary
- Test with Rich Results Test tool
