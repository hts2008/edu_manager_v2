# Structured Data

## JSON-LD Implementation
- Place in head section of HTML
- Use @context: https://schema.org
- One primary type per page

## Common Schema Types
- Organization: company info, logo, social links
- Product: name, price, availability, reviews
- Article: headline, author, datePublished
- FAQ: question/answer pairs for rich snippets
- BreadcrumbList: navigation path
- LocalBusiness: address, hours, reviews

## Testing
- Google Rich Results Test for validation
- Schema Markup Validator for completeness
- Monitor in Google Search Console

## Best Practices
- Mark up only visible content
- Keep data accurate and updated
- Dont mark up content that doesnt exist on page
