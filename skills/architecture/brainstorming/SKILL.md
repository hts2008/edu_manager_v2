# Brainstorming

## Diverge Phase
- Generate ideas without judgment (quantity over quality)
- Build on others ideas (yes, and)
- Explore extreme solutions (10x, 0.1x)
- Time-boxed: 15-30 minutes

## Converge Phase
- Group similar ideas into themes
- Dot-vote on top candidates
- Evaluate against constraints (time, budget, skill)
- Select top 2-3 for deeper analysis

## SCAMPER Technique
- Substitute: replace component with alternative
- Combine: merge two features into one
- Adapt: borrow solution from another domain
- Modify: change size, shape, frequency
- Put to other use: repurpose existing code
- Eliminate: remove non-essential features
- Reverse: flip the workflow or user flow

## Decision Matrix
- List options as rows, criteria as columns
- Weight criteria by importance (1-5)
- Score each option per criterion (1-5)
- Multiply and sum for weighted total
