# Frontend Design

## Visual Hierarchy
- Size/Weight: 48-72px display, 24-36px heading, 14-16px body, 12px caption
- Contrast: high contrast attracts attention, use selectively
- Spacing: consistent scale 4,8,12,16,24,32,48,64px

## Color Theory
- Complementary (high contrast), Analogous (harmony), Triadic (vibrant)
- 1 primary + 1-2 accents + neutral palette + semantic colors
- WCAG AA: 4.5:1 normal text, 3:1 large text

## Typography
- Sans-serif for UI (Inter, Roboto), Serif for editorial, Mono for code
- Max 2 font families. Type scale with 1.25 ratio
- Line height: headings 1.1-1.3, body 1.5-1.75
- Letter spacing: headings -0.02em, ALL CAPS +0.05em

## Layout
- 12-column grid standard. CSS Grid 2D, Flexbox 1D
- More whitespace = more premium. Padding 24-48px, section margin 48-96px
- Left-align body text. Center short headings and CTAs

## Animation
- 150-300ms micro-interactions, 300-500ms transitions
- ease-out enter, ease-in exit, ease-in-out move
- Never block interaction with animation
