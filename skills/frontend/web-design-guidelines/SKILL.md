# Web Design Guidelines

## Modern Trends
- Glassmorphism: backdrop-filter blur(10px), rgba background
- Dark mode first: reduces eye strain, saves OLED battery
- Gradient mesh for backgrounds and hero sections

## Page Layouts
- Landing: Hero > Social proof > Features > How it works > Pricing > FAQ > CTA > Footer
- Dashboard: fixed sidebar 240-280px, top bar, card widgets
- Content: max-width 680-720px for readability

## Loading States
- Skeleton screens over spinners for known layouts
- Progressive loading: text before images, inline critical CSS
- Optimistic UI: show expected result immediately

## Error/Empty States
- 404: friendly message + search + popular links
- Forms: inline validation on blur, summary at top, preserve input
- Empty: illustration + message + primary action CTA

## Performance Perception
- Instant feedback under 100ms, progress bar over 1s
- Animated transitions feel faster than instant jumps
