# Brainstorming Skill

## Quick Reference

### Diverge-Converge Framework
1. **Frame**: State problem in one sentence + constraints
2. **Diverge**: Generate ≥ 3 approaches (no judgment)
3. **Evaluate**: Score each approach on weighted criteria
4. **Converge**: Select winner with documented rationale
5. **Decide**: Record in decisionLog.md

### Evaluation Matrix Template
| Weight | Criteria | Approach A | Approach B | Approach C |
|--------|----------|-----------|-----------|-----------|

## Anti-Patterns
- Analysis paralysis: evaluating forever instead of deciding
- Anchoring: first idea dominates all discussion
- Groupthink: no one challenges the dominant approach
