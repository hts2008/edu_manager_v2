## Steps

1. Understand goal: WHO benefits, WHAT they need, WHY it matters.

2. Define scope: what's IN, what's OUT, what's DEFERRED.

3. Decompose: Goal → Stories → Phases (P0→P1→P2) → Tasks.

4. Each task: one agent, one session, estimable (XS/S/M/L).

5. Map dependencies: which tasks block which? Find critical path.

6. Assign agents from `agents/registry.yaml`.

7. Create KANBAN rows: `| T-XXX | desc | agent | PLANNED | depends | priority | estimate |`.

8. Validate: no circular deps, total fits sprint, all tasks have definition of done.
