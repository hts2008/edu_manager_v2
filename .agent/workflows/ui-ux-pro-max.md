## Steps

1. Audit existing design tokens: colors, spacing, typography, shadows, border-radius.

2. Inventory components: list existing, identify gaps and inconsistencies.

3. Design/refine design system: tokens → primitives (Button, Input, Card, Modal, Toast).

4. Implement components with all variants and states:
   - Default, hover, active, focus, disabled
   - Loading, error, empty, success
   - Mobile, tablet, desktop responsive

5. Add micro-animations: hover effects, transitions, skeleton loading.

6. Accessibility pass: contrast ≥ 4.5:1, focus rings, screen reader labels, keyboard nav.

7. Visual verification: capture screenshots for each breakpoint and state.
