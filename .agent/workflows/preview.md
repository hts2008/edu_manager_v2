## Steps

1. Start dev server or build production:
   ```
   npm run dev
   ```

2. Open browser to localhost URL. Check all key pages.

3. Visual checklist:
   - Mobile (375px), Tablet (768px), Desktop (1440px) responsive
   - All states: loading, error, empty, success
   - Dark mode if implemented
   - Animations and transitions smooth

4. Capture screenshots of key screens for review evidence.

5. Report: issues found or visual quality confirmed.
